# Bibata Modern Ice Cursor - Auto Installer

One-click installer for [Bibata Modern Ice](https://github.com/ful1e5/Bibata_Cursor) cursor theme on Windows.

## Features

- Downloads the latest Bibata-Modern-Ice-Windows.zip automatically
- Extracts and installs the cursor theme with admin privileges
- Applies the cursor theme without manual confirmation
- Auto-detects system language (English / Turkish)
- Cleans up temporary files after installation

## Usage

1. Download `install-bibata-cursor.bat`
2. Double-click to run
3. Accept the admin permission prompt
4. Done! The cursor theme will be applied automatically.
